# TASK-019 — режимы отображения List / Grid в окне Assets

## Зависимость

TASK-018 завершена. Сначала выполните её проверки, включая визуальную.

## Цель

В заголовке окна Assets появляется переключатель режима отображения: **List** (текущие строки с маленьким превью) и **Grid** (сетка крупных плиток-превью). Выбранный режим переживает перезагрузку страницы.

## Зафиксированные решения

- В `useUiPrefsStore` добавляется поле `assetsViewMode: "list" | "grid"` (default `"list"`) и операция `setAssetsViewMode(mode)`. Персистится тем же механизмом, что и остальные преференсы (`pixi-ui-editor:ui-prefs`).
- Переключатель — два маленьких кнопки-toggle в заголовке окна (между заголовком и `×`), активный режим подсвечен. Иконки — символы или inline-SVG на усмотрение исполнителя, с `title` («List view» / «Grid view»).
- Режим List — существующая разметка `asset-row` без изменений.
- Режим Grid: CSS grid из плиток шириной ~96px; в плитке сверху квадратное превью (существующий `AssetPreview`, растянутый до плитки), под ним имя (с обрезкой ellipsis и `title`) и usage count. Кнопки Replace / Delete — компактные, в нижней части плитки; семантика (включая блокировку Delete по usage с подсказкой) идентична режиму List.
- Логика usage count, upload, replace, delete не дублируется: вычисления остаются общими, режимы отличаются только разметкой/CSS.
- Fallback-превью (битый asset) должен выглядеть аккуратно в обоих режимах.

## Что создать

- Поле `assetsViewMode` в `uiPrefs.ts`.
- Переключатель в заголовке окна и grid-разметку в `AssetPanel.tsx` / `styles.css`.

Новых тестов не нужно.

## Визуальная проверка

Перед проверкой загрузите через Upload 2–3 своих картинки, чтобы в библиотеке было минимум 3–4 ассета.

1. Откройте окно Assets → режим List, всё как в TASK-018.
2. Переключитесь в Grid → ассеты отображаются плитками с крупными превью, именами и usage count; плитки переносятся на следующий ряд.
3. В Grid нажмите Replace у любого ассета и выберите файл → превью плитки и картинка на canvas обновились.
4. В Grid у используемого ассета Delete заблокирован с подсказкой, у неиспользуемого — удаляет плитку.
5. Перезагрузите страницу → окно открыто в режиме Grid.
6. Переключитесь обратно в List → строки как раньше.

## Критерии приёмки

- Оба режима показывают одинаковый набор ассетов и одинаково работающие действия.
- Режим хранится в `pixi-ui-editor:ui-prefs`, не в документе.
- Шесть шагов визуальной проверки воспроизводятся.
- Обязательные команды зелёные.
